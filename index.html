Q: What is this?

         
A: This is an interpreter for the esoteric programming language brainf*ck. The interpreter is made in rust, and is compiled with wasm-pack into WebAssembly.


         
Q: What is brainf*ck?

         
A: Brainf*ck is an esoteric programming language that is recognizable from its extreme minimalism. It was created in 1993 by german swiss computer engineer Urban MÃ¼ller. While the language is turing complete, it is not intended for practical use.


         
Q: How brainf*ck works?

         
A: Brainf*ck consists of code, (see the code section). And an array of memory cells (30,000 cells). The language consists of the following characters:

         
    
            
  • + : Increment the value current cell
    • 
            
  • - : Decrement the value current cell
    • 
            
  • > : Move the selected memory cell to the right
    • 
            
  • < : Move the selected memory cell to the left
    • 
            
  • [ : Start a loop. Check if the current cells value is zero, if it is move onto the next cell, otherwise move to the end of the loop ]
    • 
            
  • ] : If the value of the current cell is NOT 0, then move to the beginning of the loop (jump to the matching [). Otherwise move forward.
    • 
            
  • . : Output current cell value as an ASCII character to the output.
    • 
            
  • , : Accept a byte of data. (NOT IMPLEMENTED YET ;) )
    •
